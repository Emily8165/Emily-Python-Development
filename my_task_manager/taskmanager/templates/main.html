Welcome to Emily's task managing site

You will be able to:

  • View Tasks
  • Add new tasks
  • Delete tasks

Later you will be able to:

  • View the change record
  • Recover deleted tasks
  • Group tasks together

  • Add new fields
  • Be able to see the actions assigned to you through a deadicated account
